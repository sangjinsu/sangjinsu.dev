# sangjinsu.dev

서버 개발자 상진수의 포트폴리오 사이트

## Tech Stack

- **Framework**: Next.js 16 (App Router + Turbopack)
- **Runtime**: Bun
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **UI**: shadcn/ui

## Getting Started

```bash
# 의존성 설치
bun install

# 개발 서버 실행
bun --bun run dev

# 빌드
bun --bun run build
```
